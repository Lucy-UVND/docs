## Lizenz einrichten


1. Besuche lucy.uvnd.tech im Browser
2. Auf mobilen Geräten öffne das Menü ganz oben links auf den drei Strichen
3. Wähle unten Einstellungen aus
4. Lizenzschlüssel eintragen. Speichern, Los geht´s!
